01 — О боте
Exchange Desk Bot

Exchange Desk Bot — мультипользовательская система управления обменными пунктами, полностью встроенная в Telegram. Автоматизирует весь цикл операций: от приёма заявки до формирования отчётности. Каждый обменный пункт имеет полностью изолированную базу данных — свои валюты, заказы, сотрудники и история.

02 — Настройка
Обменный пункт

Виртуальный обменный пункт — изолированная рабочая область: своё название, базовая валюта, часовой пояс, сотрудники, история заказов. Можно создать несколько пунктов для разных точек обмена — данные никогда не смешиваются.

⚡Архитектура системы: один пользователь может работать в нескольких обменных пунктах одновременно. Каждый обменник имеет полностью изолированную базу данных.

Создание нового обменного пункта

Настройки › Обменник › Создать обменник
1

Введите название

Придумайте понятное название: «Обменник на Ленинском», «Crypto Exchange», «Валюта 24/7». Отображается в шапке бота и в канале сотрудников.

2

Выберите базовую валюту

Базовая валюта — национальная валюта страны, в которой работает обменник. Используется как расчётная: RUB, UAH, KZT, USD, EUR и другие.

3

Выберите часовой пояс

Укажите часовой пояс для корректного отображения времени в заказах: Europe/Moscow, Asia/Almaty, Europe/Kiev, Asia/Dubai.

4

Подтверждение

Бот пришлёт сообщение с параметрами созданного обменника. Вы автоматически становитесь Владельцем с полными правами.

Валюты и пары

Список торгуемых валют

Настройки › Валютные пары › База данных валют

Валюты вводятся в одну строку через запятую. Только латиница, цифры, символы _ и -. Кросс-курсы создаются отдельно — не через слэш.

✅ USD, EUR, GBP, CHF — стандартные валюты
✅ USD_blue, USD_new, EUR500 — кастомные тикеры
❌ USD/RUB, евро, USDT💶 — такой формат не принимается
⚠️При любом добавлении или изменении списка валют все текущие котировки сбрасываются. Используйте только латиницу, нижнее подчёркивание и дефис.
💡После ввода валют бот спросит: «Добавить RUB к торгуемым валютам?» — для большинства обменников выбирайте «Нет». Базовая валюта нужна в списке только если обменник явно торгует рублями как товаром.

Кросс-валютные пары

Настройки › Валютные пары › Кросс-курсы › Добавить кросс-курс
💱 Что такое кросс-пара

Обмен двух иностранных валют, минуя базовую. Формат записи: ТОРГУЕМАЯ / БАЗОВАЯ

Торгуемая EUR
/
Базовая USD
EUR/USD = 1.08

1

Выберите торгуемую валюту

Бот покажет список уже добавленных валют. Выберите ту, которой торгуете (например, EUR) — это «товар».

2

Выберите базовую валюту пары

Выберите валюту, в которой выражена цена (например, USD) — это «ценник». Итоговая пара: EUR/USD.

3

Задайте название пары

Бот предложит стандартное название (EUR/USD). Можно оставить стандартное или задать своё:

ВариантПримерКогда использовать
СтандартноеEUR/USDБольшинство случаев
С пометкойEUR/USD (CASH)Разделить наличные / безнал
С номиналомEUR500/USD100Крупные купюры
4

Управление созданными парами

После создания в списке кросс-курсов доступно переименование пары и удаление. При удалении все котировки по этой паре и заказы со статусом «Новый» будут удалены — заказы в других статусах сохраняются в истории.

⚠️Нельзя изменить состав валют в существующей паре. Если нужно поменять — удалите пару и создайте новую. Например: была EUR/USD, нужна USD/EUR → удалите и создайте заново.

Номиналы купюр

Номиналы нужны для точного расчёта сумм. Если бот знает, какие купюры у вас есть, он предложит правильное округление.

СпособКогда использоватьПуть
Глобальная базаСтандартные номиналы (USD, EUR, GBP...)Настройки → Обменник → Глобальные валюты
ВручнуюНестандартные или ограниченные номиналыНастройки → Валютные пары → Кастомизация → Номиналы
💡Зачем нужны номиналы: клиент продаёт 7520 GBP, расчётная сумма — 9964 USD. Без мелких купюр её не выдать. Бот использует настроенные номиналы (50, 100) и предложит: округлить до 10 000 USD (доплата клиента) или выдать 9900 USD (доплата офиса).

ПараметрДоПослеРазница

Сумма EUR500.00500.00—

Изменение суммы получения. У клиента ровно 40 000 RUB — нужно пересчитать количество EUR. Нажмите «Сумма получения RUB» → введите 40000. Бот пересчитает сумму EUR по текущему курсу.

05 — Справка
Часто задаваемые вопросы
Общие вопросы
Можно ли работать одновременно в нескольких обменных пунктах?

Да! Один пользователь может быть владельцем своего обменного пункта и одновременно работать оператором в чужом. Данные разных обменников полностью изолированы — пересечений нет.

Что делать, если бот не отвечает?
  1. Проверьте интернет-соединение
  2. Перезапустите Telegram
  3. Отправьте команду /start
  4. Если проблема сохраняется — свяжитесь с поддержкой
Как удалить обменный пункт?

Удалить обменник может только владелец через настройки. Все данные будут безвозвратно удалены. Эту операцию нельзя отменить.

Вопросы по валютам
Зачем нужна связка с глобальной базой номиналов?

Глобальная база автоматически подгружает все существующие номиналы купюр для валюты. Это экономит время на ручной настройке и гарантирует актуальность данных. Без номиналов бот не может предложить корректные варианты округления при создании заказа.

Можно ли создать кастомный тикер валюты?

Да, можно создать любой тикер: USD_new, EUR500, Dollar_cash. При связке с глобальной базой бот попытается автоматически определить соответствующую мировую валюту по названию.

Как изменить дробную часть после настройки?

Настройки → Валютные пары → Кастомизация валют → Дробная часть → выбрать валюту → ввести нужное значение (2–8 знаков). Изменение вступает в силу немедленно.

Вопросы по заказам
Можно ли редактировать заказ на кросс-пару?

Функция редактирования кросс-курсов находится в разработке. Для изменения нужно отменить заказ и создать новый с корректными параметрами.

Как изменить статус заказа?

Откройте заказ на редактирование любым из трёх способов (предпросмотр / меню бота / ссылка в канале) и выберите «Изменить статус».

Вопросы по каналу
Как получить chat_id канала?

Используйте любого стороннего бота — перешлите ему пост из канала:

БотОсобенность
@JsonDumpBotПодробный JSON с полной структурой объекта
@getidsbotМного информации об объекте
@myidbotКлассика — простой и надёжный
@userinfobotID при пересылке сообщений
@getmyid_botБыстро и просто

Алгоритм: Откройте бота → нажмите Start → перешлите любой пост из канала → получите chat_id.

🚫Chat_id канала всегда содержит знак - в начале. Копируйте вместе со знаком минус! Например: -1003639640960
Обязательно ли создавать канал для сотрудников?

Нет, это опционально. Бот работает и без канала. Но канал значительно упрощает командную работу: все заказы видны в единой ленте, можно обсуждать и редактировать прямо там.

Можно ли использовать публичный канал?

Настоятельно рекомендуется использовать закрытый (приватный) канал для конфиденциальности данных клиентов и операций.

Почему бот не может публиковать в канал?

Проверьте три вещи:

  1. Бот добавлен как администратор канала
  2. У бота есть права на публикацию сообщений
  3. Chat_id канала введён правильно (со знаком -)
